More About Me

My journey into web development began with learning the fundamentals of HTML and CSS. As my interest grew, I expanded my knowledge by learning JavaScript and Bootstrap to create interactive and responsive websites. Every project I complete helps me improve my problem-solving skills and strengthens my understanding of modern web development.

I am currently pursuing a Bachelor's degree in Computer Science (BSCS), where I continue to develop my knowledge of programming, software engineering, and web technologies. Alongside my academic studies, I regularly build practical projects to improve my technical skills and gain hands-on experience.

I have a solid foundation in HTML, CSS, JavaScript, and Bootstrap. I enjoy creating responsive layouts, smooth animations, interactive user interfaces, and clean website designs. I am committed to writing organized, maintainable code while continuously learning modern development practices.

My primary goal is to become a professional Full Stack Web Developer. I plan to expand my expertise by learning React, Node.js, databases, and backend development. I believe continuous learning and practical experience are the keys to becoming a successful software developer.

I believe every project deserves attention to detail, clean code, and a user-focused design. I am committed to delivering responsive, fast, and visually appealing websites while maintaining clear communication and a professional approach throughout every project.

I am continuously improving my skills by exploring advanced JavaScript concepts, responsive web design techniques, Git and GitHub, and modern frontend technologies. My focus is on staying updated with the latest industry trends and best practices.

Thank you for taking the time to learn more about me. I am always excited to work on new ideas, collaborate with clients, and build high-quality web experiences. If you have a project in mind or would like to discuss an opportunity, feel free to get in touch. I would be happy to connect with you.
